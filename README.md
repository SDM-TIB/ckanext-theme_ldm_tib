# ckanext-theme_ldm_tib

This CKAN extension provides the theme of the main instance of the [Leibniz Data Manager](https://service.tib.eu/ldmservice/) (LDM) as hosted by the [Technische Informationsbibliothek](https://www.tib.eu/) (TIB).

## Requirements

This extension has been tested with CKAN 2.9.

## Installation

As usual for CKAN extensions, you can install `ckanext-theme_ldm_tib` as follows:

```bash
git clone git@github.com:SDM-TIB/ckanext-theme_ldm_tib.git
pip install -e ./ckanext-theme_ldm_tib
```

After installing the plugin, add `TIBtheme` to the plugins in your `ckan.ini`.
